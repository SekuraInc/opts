## Usage

#### Initialize a new Cafe

```
$ ./init.sh -k <key_file> -r <release_tag> -p <public_ip> -u <https_url>
```

#### Upgrade a Cafe

```
$ ./upgrade.sh -k <key_file> -r <release_tag> -p <public_ip>
```

#### Upgrade all Cafes

```
$ ./upgrade-all.sh -r <release_tag>
```
